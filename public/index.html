Configuration Meta WhatsApp

Comment obtenir ces informations ?
  1. Allez sur developers.facebook.com
  2. Créez ou sélectionnez votre application
  3. Ajoutez le produit "WhatsApp" à votre app
  4. Dans WhatsApp > Configuration de l'API vous trouverez :
    • Token d'accès temporaire (ou créez un token permanent via les paramètres système)
    • Phone Number ID (ID du numéro)
    • WhatsApp Business Account ID
  5. Pour le Webhook, configurez l'URL de callback : https://VOTRE_DOMAINE/webhook avec le token de vérification ci-dessous

Templates WhatsApp

Comment fonctionnent les templates ?

Les templates sont des messages pré-approuvés par Meta. Vous devez les créer et faire approuver sur business.facebook.com avant de pouvoir les utiliser pour contacter des prospects.

Cliquez sur "Synchroniser depuis Meta" pour importer automatiquement vos templates approuvés.

Campagnes

Comment lancer une campagne ?
  1. Assurez-vous d'avoir des templates approuvés (onglet Templates > Synchroniser depuis Meta)
  2. Ajoutez vos contacts (onglet Contacts)
  3. Créez la campagne ci-dessous en sélectionnant un template et les contacts
  4. Cliquez sur "Lancer" pour envoyer les messages

Note : Seuls les templates approuvés par Meta peuvent être utilisés pour contacter des prospects en dehors de la fenêtre de 24h.
