Configuration Meta WhatsApp

Comment obtenir ces informations ?
  1. Allez sur developers.facebook.com
  2. Créez ou sélectionnez votre application
  3. Ajoutez le produit "WhatsApp" à votre app
  4. Dans WhatsApp > Configuration de l'API vous trouverez :
    • Token d'accès temporaire (ou créez un token permanent via les paramètres système)
    • Phone Number ID (ID du numéro)
    • WhatsApp Business Account ID
  5. Pour le Webhook, configurez l'URL de callback : https://VOTRE_DOMAINE/webhook avec le token de vérification ci-dessous
Identifiants Meta
Token permanent recommandé pour la production. Le token temporaire expire au bout d'une heure.
Identifiant du numéro de téléphone WhatsApp Business.
ID du compte WhatsApp Business (WABA).
Chaîne secrète que vous définirez aussi dans la config webhook Meta.
Configuration du Webhook
URL du Webhook à renseigner dans Meta :

https://VOTRE_DOMAINE/webhook

Pour exposer votre serveur local, utilisez ngrok http 3000 puis copiez l'URL HTTPS générée.
